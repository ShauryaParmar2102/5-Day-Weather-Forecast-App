# 5-Day-Weather-Forecast-App

A responsive weather web application that provides **current weather conditions** and a **5-day forecast** using the OpenWeatherMap API. Users can search for any city or use their current location to get real-time weather data.

---

## 🚀 Features

- 🔍 Search weather by city name
  
- 📍 Get weather using your current location (GPS)
  
- 🌡️ Current temperature, humidity, and wind speed
   
- 📅 5-day weather forecast
  
- 🌤️ Weather icons and descriptions
  
- ⚡ Fast API-based data fetching
  
---

## 🛠️ Tech Stack

- HTML
  
- CSS

- JavaScript
 
- OpenWeatherMap API

## ⚙️ How It Works

1. User enters a city OR clicks location button
   
2. App gets latitude & longitude using OpenWeather Geocoding API
  
3. Weather data is fetched using those coordinates
   
4. UI updates with:
   - Current weather
   - 5-day forecast cards
